# 全局化配置 Config Provider

## 演示

```demo
cls-prefix
```

## Props

| 名称       | 类型      | 默认值  | 说明                                     |
| ---------- | --------- | ------- | ---------------------------------------- |
| abstract   | `boolean` | `false` | 是否不存在 DOM 包裹                      |
| cls-prefix | `string`  | `d`     | 内部所有组件的类的前缀，仅首次设定会生效 |
| tag        | `string`  | `'div'` | `n-config-provider` 被渲染成的元素       |
